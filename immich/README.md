# Immich Examples

Example configurations and management scripts for [Immich](https://immich.app/) - the self-hosted photo and video management solution.

## Examples

| Example | Description |
|---------|-------------|
| [windows/](./windows/) | Windows setup with external library, auto album creation, and management scripts |
| [tailscale/](./tailscale/) | Remote access to Immich via Tailscale Docker sidecar with HTTPS |
| [cloudflare/](./cloudflare/) | Public access to Immich via Cloudflare Tunnel for sharing with anyone |

## Contributing

Feel free to add examples for other platforms or configurations:
- Linux server setup
- Synology NAS
- Raspberry Pi
- Kubernetes/Helm
- etc.

Each example should be self-contained in its own directory with its own README.
